# Как использовать 
---

1. Скачать [docker-compose](https://docs.docker.com/compose/install/) и [git](https://git-scm.com/install/)

2. Клонировать репозиторий
```bash
git clone https://github.com/Timokrut/metadata_analysis.git
cd metadata_analysis
```

3. Сбилдить и поднять образ
```bash
docker-compose up -d --build
```

4. Приложение будет доступно по ссылке: 
```bash
http://localhost:8003
```
